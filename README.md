# Personal Website — Robert R. Leung

A clean, academic personal site built as a static page for **GitHub Pages**.

## Files
- `index.html` — the site
- `style.css` — styles
- `assets/cv.pdf` — CV (replace with your final version)
- `.nojekyll` — tells GitHub Pages to serve files as-is

## Publish on GitHub Pages
1. Create a repository named `<your-username>.github.io` (this makes the site live at that URL).
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**, set **Source** to `Deploy from a branch`, branch `main`, folder `/ (root)`, and save.
4. Your site goes live at `https://<your-username>.github.io` within a minute or two.

## Update the CV
Replace `assets/cv.pdf` with your final (law-school) CV, keeping the same filename so the link keeps working.
